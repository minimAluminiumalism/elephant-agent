# Companion Scenarios

This directory seeds scenario fixtures for the text-first `companion` profile path.

Track goals:

- canonical identity state stays visible and inspectable
- first-run onboarding can distinguish defaults from an explicitly confirmed companion identity
- relationship continuity survives resume and interruption
- text-first persona state can later extend to voice without changing identity

Canonical downstream consumers:

- `apps/cli/**`
- `packages/profile/**`
- `packages/session/**`
