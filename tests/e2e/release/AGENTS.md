# Release Certification E2E Tests

Use this directory for project-ready certification flows that prove the shipped
install, setup, provider, continuity, and deploy path can be signed off by the
main session.

Good targets:

- deterministic release matrices
- manual live-provider smoke paths
- release-captain checklists

Do not own here:

- provider runtime internals
- surface-specific business logic already covered by other e2e suites
